Arctic Web Map

About Arctic Web Map

AWM is a component in the Arctic Connect project. It is designed to provide a better mapping tool for researchers by displaying a more appropriate projection for Arctic research.

AWM is built on an open source mapping stack, using similar technologies to OpenStreetMap. Background map data is used under license from OpenStreetMap contributors. If you see an error in the map data for Arctic Web Map tiles, you can make updates to the OpenStreetMap web site and the updates will be reflected in our tiles after a short (less than an hour) update period.

If you have any questions or want to learn more about Arctic Web Map or Arctic Connect, please contact the project PI: Steve Liang.

If you have found a bug or the tiles/client library do not work as you expect, please contact the project technical manager: James Badger.
